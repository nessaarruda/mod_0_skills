class Plate

Attributes

shape (string)
weight (float)
color (string)
material (string)

Methods

check_shape (modifies the attribute `shape` by checking if the shape is round, if it is mark as "lunch plate")
subtract_weight (modifies the attribute `weight` by subtracting the weight of the plate to return only the weight of the food)
change_color (modifies the attribute `color` by checking if the color is red if it isn't change the color to red)
check_material (modifies the attribute `material` by checking if the material is glass, if it is mark as "dinner plate")
